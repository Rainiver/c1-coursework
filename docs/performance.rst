Performances and Profiling
==========================

Performance Analysis
--------------------

This section presents the performance analysis of the model as a function of dataset size (1K, 5K, 10K).

**Scalability Findings:**
- **Training Time:** Expected linear or sub-linear scaling with batch processing.
- **Memory Usage:** Increases with dataset size, constrained by batch size.
- **Accuracy (MSE/R²):** Generally improves with more data, plateauing at model capacity.

*Note: Run the profiling script to generate actual data.*
